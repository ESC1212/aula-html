2. Leia a frase e comente com completude e coerência.

“ HTML é uma linguagem de programação de uso geral, assim como CSS, java e javascript.”

Html CSS e java não são a mesma coisa
html é uma linguagem feita para mandar arquivos de texto pela web, é usada para fazer o layout de um site, não é possivel aplicar qualquer tipo de logica de programação
CSS é uma linguagem feita para estilizar uma página, não é possivel criar uma pagina web somente com CSS
java e javascript sim é uma linguagem de uso geral, é possivel aplicar logica de programação.
