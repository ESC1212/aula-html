C. O que é um tag HTML?

as tags são uma forma de estruturar a pagina html. ex: h1, p, body, etc...

D. O que é HTML e qual a estrutura básica de um documento HTML?

HTML é uma arquivo de texto que descreve a estrutura de uma pagina Web, um documento html é recomendado ter o "!DOCTYPE html" no inicio, seguido pela tag "html", ter um titulo com a tag "title", e um corpo "body".
